
# has-a
- assocation : 연관 (ref 유지), 직접적 관계 없음 '-'
- composition : 완전한 소유, 생성과 소멸을 같이 한다.(life cycle이 같다.)
- aggregation : 집합, 생성과 소멸이 같지 않다.  (life cycle이 다르다.)
- dependency : 의존 (ref 유지 안함)
# is-a
- generalization : 일반화, 상속
- realization : 전문화, 인터페이스 구현
