# 선언적으로 생각하기
## 라자냐에 대한 조작적 정의 (의존관계)
1. 기름을 팬에 달군다
2. 양파를 볶는다
3. 소고기와 토마토를 추가한다...

## 라자냐에 대한 개념적 정의
라자냐는 치즈소스 파스타 볼로네즈로 이루어진다.
볼로네즈는 양파를 갈색이 될 때까지 볶는다
치즈소스는 우유와 치즈를 추가하면서 볶는다
루는 밀가루와 버터를 볶는다
굽는 것은 오븐에 넣는다
볶는 것은 팬에 넣고 섞는다
