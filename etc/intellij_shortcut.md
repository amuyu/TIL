
# template
sout : println
psvm : main method

# editor
ctrl + alt + o : optimize import
command + alt + l : reformat
⌥⌘L : correct indention
⌥⌘M : Extract method
⌥⌘V : Introduce variable
⌥⌘F : Introduce field
⌥⌘C : Introduce constant
⌥⌘P : Introduce parameter
⌥⌘T : surround with

# open
⌘E : 최근 실행했던 파일
⌥F12  : 터미널 오픈

# search
alt + F7 : 어떤 항목이 사용된 위치 검색
⌘B : 선언으로 이동
⌥⌘B : 구현으로 이동
⇧⇧ : 찾기

# move
⌘ + ⇧ + [ : 탭의 왼쪽으로 이동 : 
⌘ + ⇧ + ] : 탭의 오른쪽으로 이동 : 
⌘ + b : 메소드 구현체로 이동

# build
shift + command + i : load build.gradle

# 줄이동
Option(⌥) + Shift + ↓
