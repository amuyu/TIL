
첫 줄에 50자 이내로 커밋 내용을 요약

50자 요약으로 커밋을 설명하기에 부족하면 두번째 줄을 비우고 세번째 줄부터 상세 내용을 적는다. 세번째 줄부터는 한 줄을 72자 이내로 제한
```log
UserCommand 추상 기반 클래스를 추가한다

다음 명령 클래스가 UserCommand 클래스를 상속하도록 변경한다.
- CreateUserWithEmail
- ChangeUsername
- ChangeEmail
```
